# How to Run AI Fitness Trainer

## Prerequisites
- Node.js installed
- Python installed

## 1. Start the Backend
Open a terminal in the `ai-fitness-trainer` folder and run:
```bash
uvicorn backend:app --reload --host 0.0.0.0 --port 8000
```

## 2. Start the Frontend
Open a **new** terminal in the `ai-fitness-trainer/frontend-react` folder and run:
```bash
npm run dev
```

## 3. Open in Browser
Go to: [http://localhost:5173](http://localhost:5173)
